# 1to9calc
Search for arifmetic operators
Ищет последовательность опреаций для чисел 1..9 для получения все чисел от 1 до 15000
Например:
1=  1^23456789
2= 1^234567-8+9
3= 1*2*345-678-9
4= 12-34-56-7+89
5= 1*2+34+56-78-9
6= 1^2345-67+8*9
7= 1^23456+7+8-9
8= 1^23456*7-8+9
9= 1^2345678*9
10= 1^2345678+9
